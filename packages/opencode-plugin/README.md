# NodeSource AI Skills for OpenCode

N|Solid performance & security skills for Node.js.

## Install

Add to your `opencode.jsonc`:

```jsonc
{
  "plugin": ["@nodesource/opencode-plugin"]
}
```

Or copy `index.js` to `~/.config/opencode/plugins/`.

## What's Included

- 15 AI skills for Node.js performance and security analysis
- MCP server integration for live production data
- Automatic authentication with NodeSource accounts
